# iAwake
It's demo of music player app that allow to get list of free programs, discover and listen available tracks.
## Project stack
- Kotlin
- Clean Architecture
- MVVM
- Koin
- Room
- Coroutines
- ExoPlayer
- Lottie
- Coil
## Roadmap
- UI/UX Improvements
- UI for landscape
- Dark and Light themes
- Details of currently played track
